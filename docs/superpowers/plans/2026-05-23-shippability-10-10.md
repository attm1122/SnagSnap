# SnagSnap Shippability 10/10 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Bring SnagSnap from buildable prototype to a shippable iOS app with green tests, verified core workflows, clean project structure, and documented release risks.

**Architecture:** Keep the existing SwiftUI + SwiftData structure. Stabilize the app by replacing stale tests with current API coverage first, then tighten runtime workflows around reports, issues, photos, PDF generation, and subscriptions.

**Tech Stack:** SwiftUI, SwiftData, XCTest, PDFKit, PencilKit, StoreKit, Xcode `xcodebuild`.
